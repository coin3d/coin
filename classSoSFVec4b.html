Detailed Description

The SoSFVec4b class is a container for an SbVec4b vector.

This field is used where nodes, engines or other field containers need to store a single vector with four elements.

See also
SbVec4b, SoMFVec4b Be aware that this class is an extension for Coin, and it is not available in the original SGI Open Inventor v2.1 API.
Since
Coin 2.5
