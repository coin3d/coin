Detailed Description

The SoMFUShort class is a container for unsigned short integer values.

This field supports application data sharing through a setValuesPointer() method. See SoMField documentation for information on how to use this function.

This field is used where nodes, engines or other field containers need to store a group of multiple short unsigned integer values.
