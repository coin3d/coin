Detailed Description

The SoMFVec4ub class is a container for SbVec4ub vectors.

This field is used where nodes, engines or other field containers need to store an array of vectors with four elements.

This field supports application data sharing through a setValuesPointer() method. See SoMField documentation for information on how to use this function.

See also
SbVec4ub, SoSFVec4ub Be aware that this class is an extension for Coin, and it is not available in the original SGI Open Inventor v2.1 API.
Since
Coin 2.5
